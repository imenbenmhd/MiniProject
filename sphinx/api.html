Generate a new feature matrix consisting of all polynomial combinations of the features with degree less than or equal to the specified degree. :param X: A 3D numpy ndarray in which the rows represent examples while the

columns, features of the data set you want to normalize. Every depth corresponds to data for a particular class.

Parameters

  • scale (string; determines whether MinMax or Z-normalization should be applied.) –

  • degree (int or tuple (min_degree, max_degree), default = number of features in X) – If a single int is given, it specifies the maximal degree of the polynomial features. If a tuple (min_degree, max_degree) is passed, then min_degree is the minimum and max_degree is the maximum polynomial degree of the generated features.

  • interaction (If True, only interaction features are produced: features that are products of) – at most degree distinct input features Desired range of transformed data.
